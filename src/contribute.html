Open Contribution

Add Your Project

Contributing to the NEU catalogue is done through GitHub pull requests. Follow the steps below to list your project.

1

Fork the Repository

Navigate to the NEU GitHub repository and click Fork to create your own copy. Then clone your fork locally:

$ git clone https://github.com/YOUR_USERNAME/NEU.git
$ cd NEU
2

Add Your Project Image

Place a clear photograph or diagram of your project in the assets/images/ directory. Use a descriptive, kebab-case filename (e.g. my-lick-detector.webp). Supported formats: .webp, .png, .jpg.

$ cp ~/photos/my-project.webp assets/images/my-project-name.webp
3

Add Your Entry to the Database

Open database.json and append a new object to the array. Follow the schema below exactly:

{
  "id": "my-project-name",
  "name": "My Project Name",
  "category": "Sensors/Touch Sensors",
  "description": "A brief description of your project.",
  "authors": ["Your Institute"],
  "image": "assets/images/my-project-name.webp",
  "link": "https://github.com/org/repo",
  "tags": ["capacitive", "lick detection"],
  "specs": {
    "Type": "Capacitive",
    "Response Time": "<1 ms"
  }
}
Field Type Description
id string Unique kebab-case identifier (e.g. my-lick-detector)
name string Human-readable project name
category string Slash-delimited category path (e.g. Sensors/Touch Sensors). Supports unlimited depth.
description string Brief description of the project and its application
authors string[] List of contributing institutions or individuals
image string Relative path to the project image in assets/images/
link string URL to the project repository or product page
tags string[] Free-form keywords for search (e.g. capacitive, fiber photometry)
specs object Key-value pairs for technical specifications (e.g. "Wavelength": "400–700 nm")
4

Submit a Pull Request

Commit your changes, push to your fork, and open a pull request against the main branch of the NEU repository. An administrator will review your submission and, if everything follows the guidelines, merge it into the catalogue.

$ git add database.json assets/images/my-project-name.webp
$ git commit -m "Add my-project-name to catalogue"
$ git push origin main
Guidelines

Submission Requirements

Tested in Neuroscience

Your project must have been used or validated in a neuroscience experiment. We do not list untested prototypes.

Documentation

A link to documentation, a repository, or a product page must be provided so others can reproduce or purchase the tool.
